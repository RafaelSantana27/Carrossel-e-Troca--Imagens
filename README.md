**🎡 Carrossel de Imagens & Troca de Imagem com HTML, CSS e JavaScript**

Este projeto tem como objetivo criar uma aplicação interativa com um carrossel de imagens e a funcionalidade de troca de imagem ao clicar em um botão. Utilizando HTML, CSS e JavaScript, o projeto é perfeito para quem está iniciando no desenvolvimento front-end e quer aprender como manipular eventos e alternar entre elementos de maneira simples e eficiente.

**📜 Descrição:**
Neste repositório, você encontrará o código para:

**Troca de imagens:** Alterna entre duas imagens com um botão.

**Carrossel de imagens:** Navegação entre várias imagens utilizando botões de "anterior" e "próximo".

**Troca automática de imagens:** Uma funcionalidade para alternar as imagens automaticamente com base em um intervalo de tempo.

**🚀 Tecnologias Utilizadas:**

+ HTML5: Estruturação da página e layout básico.
+ CSS3: Estilização das imagens, botões e layout da página.
+ JavaScript: Manipulação do DOM para alternar imagens e implementar a troca automática.
+ Font Awesome: Utilização de ícones para os botões de navegação.

**🔧 Funcionalidades**

+ Troca de imagem ao clicar no botão: Uma imagem é alternada por outra ao clicar em um botão.
+ Carrossel de imagens: Avança e retrocede entre as imagens ao clicar nos botões de navegação.
+ Alternância automática: As imagens trocam automaticamente a cada 5 segundos quando a opção de "imagens automáticas" é ativada.
